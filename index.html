Welcome!

This is a mind game where you try to guess what letter the computer is thinking!

To play:
1. Think of a letter and press it on the keyboard.
2. If you think correctly, you'll earn a score.
3. You have 10 chances to guess in a single round.
4. If you use up all your chances, you lose.
